Privacy Policy

This Privacy Policy describes how StomBee Puzzle Book Creator (the "App") handles data. The App is used solely by the owner of the Etsy shop StomBee to create and update draft listings for digital puzzle books.

Data Collected

  • OAuth tokens from Etsy are stored securely to allow the App to connect to the owner’s Etsy shop.
  • No buyer information or personally identifiable information (PII) is collected.

Data Sharing

  • No data is sold, leased, transferred, or shared with third parties.
  • The App is strictly private-use for the shop owner and not available to other users.

Data Retention & Security

  • OAuth tokens are stored securely and rotated/revoked if compromised.
  • No customer PII is stored. Listing data is the shop owner’s own content.
  • Data is retained only as long as needed to manage draft listings.

Use of Etsy API

The App uses only documented endpoints provided by Etsy and complies with Etsy’s API Terms of Use.

Audience

Only the shop owner uses this App. It is not distributed publicly or offered as a service.
