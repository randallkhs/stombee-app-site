StomBee Puzzle Book Creator

StomBee Puzzle Book Creator is a private tool built exclusively for managing my Etsy shop StomBee . The app streamlines the creation of digital puzzle book listings by combining puzzle themes, generating optimized listing titles and descriptions, and publishing draft listings directly to Etsy using the official Etsy Open API.

What this app does

  • Create draft listings in my own shop with accurate metadata.
  • Generate titles, descriptions, and keywords using my puzzle themes.
  • Upload listing images (covers/mockups) that I generate locally.
  • No access to orders, payments, or customer PII.
